# Pilot 4 — Meda Monsta (Base App)

This repository is a Pilot app built from **Base Starter Pack V1**.

## Identity
- Pilot: **4**
- Creator: **Meda Monsta**
- Scope: **Links + simple sections only**
- Rules: **No logins • No tracking • No algorithm**

## What this is
A creator-owned, installable web app (PWA) that gives fans one direct place to go.

## Update rule (important)
When changes are made, bump the cache version inside:
- `service-worker.js` (`CACHE_VERSION`)

## Files
- `index.html` — UI + tabs
- `content.json` — creator links/content
- `manifest.json` — app install identity (name/icons/colors)
- `service-worker.js` — offline + update behavior
- `assets/` — images/icons/logos
